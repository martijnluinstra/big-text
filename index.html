About BigText

BigText is a browser-side tool to create and share digital posters. It was made in 2022 by me, Martijn Luinstra, as an experiment and exercise in some modern web technology I hadn't used before. The source is available on GitHub.

If you find any mistakes, typos, or bugs, please don't hesitate to contact me. (Or submit a pull request on GitHub.)
